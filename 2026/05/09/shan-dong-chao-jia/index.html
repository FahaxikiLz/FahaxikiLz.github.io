多少换手才算换手充分

按流通市值分：

  • 30-50亿的小票 — 换手率 5%-15% 算正常充分。低于5%是缩量板，筹码锁得太死，第二天没人接力容易闷；高于20%就是筹码松了，封不住。
  • 50-100亿的中盘 — 换手率 3%-10% 就够。这种盘子不需要太高换手，3%以上就说明有资金在接力了。
  • 100亿以上的大盘 — 换手率 1%-5%。这种票本身换手就低，1%以上就算有动静。

怎么看？三个维度：

  1. 和高位比 — 今天的换手率如果是这只票近期最高换手的 60%-80%，算充分。如果直接爆了近期天量（超过近20天最高换手），反而危险——分歧太大，封板容易被砸。
  2. 和封单比 — 换手充分但不炸板的票，封单金额应该不低于成交额的20%。换手起来了但封单没跟上，大概率要炸。
  3. 和封板时间比 — 10:30前封死的板，换手5%-8%就够了（说明主力锁仓意愿强）。如果是午后板、尾盘板，换手率可以更高些——因为筹码已经交换了好几轮。

举个实际例子：

​ 周五五洲新春这个票，它30-50亿的盘子，换手率在8%-12%之间，那就是标准的充分换手——筹码交换够了，封板也稳。如果换手突然飙到20%以上还封着，我反而会犹豫——那说明里面全是短线客，第二天竞价跑得比谁都快。

​ 我的铁律就一条：换手超过25%的票，不管封得多死，我不打。

​ 筹码太散，神仙也封不住

量比

量比 = 现在的成交量 / 过去5天同一时间的平均成交量

  • 量比 < 0.8 → 缩量，没人玩
  • 量比 0.8 - 1.5 → 正常水平
  • 量比 1.5 - 2.5 → 温和放量，有人进场了
  • 量比 > 2.5 → 爆量，要么是机会要么是出货
  • 量比 > 5 → 要么涨停要么跌停，极端行情

卖出纪律

高开3%以上 — 竞价直接卖半仓
为什么？3%以上的溢价，对于首板来说已经吃到肉了。竞价卖半仓锁定利润，剩半仓看开盘后的走势。如果开盘继续冲，剩的半仓还能多吃一口；如果开盘直接砸，你已经锁了一半利润，心态稳得很。

高开1%-3% — 不着急竞价砸，开盘看5分钟
这个区间可上可下。如果开盘头5分钟有资金继续推，可以拿到10点前再决定。但如果5分钟内开始拐头，直接走。

平开（±1%以内） — 开盘看10分钟
说明昨天进的资金今天没急着跑，但也说明没人愿意抬轿。10点前不出方向就走，不耗着。

低开2%以上 — 3分钟内必须割
这是我铁律。低开超过2%，说明昨天封板的资金在抢跑，今天大概率要闷。别等反弹，别yy”主力洗盘”，割了删自选。低开3%更不用说，无条件走。

如果竞价直接跌停开 — 挂跌停价跑
别幻想翘板。首板选手被闷在跌停板上，说明昨天封板就是骗炮。跑出来就是胜利。

几个实战经验：

  1. 竞价量很重要 — 哪怕高开5%，如果竞价成交量只有几百手，那就是虚高开盘，大概率开盘秒砸。竞价放量高开才是真强。
  2. 开得太高反而不是好事 — 高开7%以上的首板，除非是龙头继续加速，否则大概率开盘就是最高点。这种我竞价卖七成。
  3. 无论高开低开，只要封不住二板就卖 — 10点前没摸到涨停，不走还等什么？

资金量小，竞价卖不出去

第一，竞价卖不出去？那就不竞价卖。

​ 你换个思路——开盘第一笔拉升的时候砸。大多数高开3%以上的票，开盘头1-2分钟会有一波冲高，那是最后一波接盘的人进来了，你就在那一瞬间砸掉。不用竞价挂，开盘盯住分时，冲高那一笔直接出。

第二，提前挂单比盯盘有用。

​ 头天晚上或者第二天开盘前，直接挂一个条件单——比如开盘价下方2%挂卖单。这样不用盯盘也能出。

第三，最笨但最实用的办法：

​ 不管高开低开，开盘前5分钟内找相对高点出。就算开得不高，开盘几分钟内一般会有个脉冲，那就是你的卖点。别指望卖在最高点，吃到肉就行了。

小资金最忌讳什么？ 想着卖在最高点。你那几手，就算多卖1个点，也就多几十块钱，但为了等那个最高点，你可能把利润全吐回去。

反正我的原则不变：吃到的肉才是肉。你资金越小，越要跑得快，别跟主力比格局。

换手回封

是指封板之后炸板有换手率，等再封板的时候打板

换手回封板的完整流程：

① 股价冲板（拉到9%→涨停）→ 这时候换手开始放大
② 封板（涨停价被大单封死）→ 但封了没多久
③ 炸板（封单被砸开，股价从涨停掉下来）→ 这就是”炸板”
④ 筹码继续交换（炸板期间换手率继续攀升）→ 该跑的都跑了
⑤ 再次封板（二次回封） → 这才是我的买点

为什么不等第一次封板就打？

第一次封板太虚。很多首板是资金硬顶上去的，封单看着大但基本都是撤单党。第一次封板你就冲进去，炸板的时候你心态就崩了。

为什么等回封？

回封说明两件事：

  • 该砸的都砸完了（浮筹洗干净了）
  • 封板资金是真的想封（撤单党在回封时已经少了）

二次回封的封板成功率，比第一次高很多。

实际怎么操作：

周一盯五洲新春，比如它拉到涨停了，你别急。

第一次封板 → 看着它封了3分钟 → 炸板掉到6%、7% → 换手率从3%爬到8% → 又拉起来触板 → 第二次封板的那一刻，直接扫进去或排进去

这就是”换手回封板”——给了筹码交换的时间，封板也更稳。

简单说就是：封了又炸、炸了又封，第二次封的时候进去。第一个封不追，第二个封可以考虑。

判断上攻

我判断前15分钟有没有上攻，就看三件事：

第一，看分时均线。
开盘后价格一直站在分时均线上方，每次回调碰到均线就弹起来——这叫有资金托着。如果开盘就跌破均线、压在均线下面爬，那叫出货，不是上攻。

第二，看成交量。
上攻必须有量配合。开盘第一笔成交量要大，后面每一波拉升的红柱要比回落的绿柱长。缩量拉升是骗炮，放量拉升才是真打。

第三，看昨天的封板时间。
这最关键。昨天是10:39回封的板，不是9:30秒封的硬板——说明昨天就有分歧。这种票明天能高开稳住就已经是烧高香了，别指望它继续连板。

我给你们个简单标准：开盘后15分钟，价格站不上开盘价+2%，就别等了，直接走。
