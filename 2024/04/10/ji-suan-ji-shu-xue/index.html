数列

等差数列

  1. 通项公式：$$a_n = a_1 + (n - 1) \times d$$
  2. 前n项和：$$S_n = \frac{n \times (a_1 + a_n)}{2}$$

等比数列

  1. 通项公式：$$a_n = a_1 \times r^{(n-1)}$$
  2. 前n项和：$$S_n = \frac{a_1 \times (1 - r^n)}{1 - r}$$

反函数

反函数是指对于一个函数 f(x)，如果存在另一个函数 g(y)，使得对于所有的 x 和 y，满足 g(f(x)) = x 和 f(g(y)) = y，那么函数 g(y) 就被称为函数 f(x) 的反函数。简单来说，如果对于函数 f(x) 的每个输出 y，存在唯一的输入 x，使得 f(x) = y，则函数 f(x) 的反函数就是将 y 作为输入，输出对应的 x。

让我们考虑一个简单的函数，例如 $f(x) = 2x + 3$。我们可以找到它的反函数。

首先，我们将 $f(x)$ 表示为 $y$，即 $y = 2x + 3$。

接下来，我们通过交换 $x$ 和 $y$ 来解出 $x$。这样，我们就可以得到反函数了。

将 $y = 2x + 3$ 中的 $y$ 替换为 $x$：$x = 2y + 3$。
解出 $x$：$x - 3 = 2y$，然后除以 $2$：$\frac{x - 3}{2} = y$。
所以，反函数是 $g(y) = \frac{x - 3}{2}$。

验证一下：

对于函数 $f(x)$，我们有 $f(4) = 2(4) + 3 = 11$。
现在，我们将 $11$ 作为输入，使用反函数，$g(11) = \frac{11 - 3}{2} = 4$。
所以，$f(4) = 11$，同时 $g(11) = 4$，这证明了这两个函数互为反函数。

指数和对数

互为反函数

函数 $f(n) = 2^n$ 是一个指数函数，你想找到它的反函数。反函数可以表示为对数函数。对于这个情况，反函数是以 2 为底的对数函数。

反函数可以表示为 $g(x) = \log_{2}(x)$。

验证一下：

  1. 对于函数 $f(n) = 2^n$，当 $n = 3$ 时，$f(3) = 2^3 = 8$。
  2. 现在，我们将 $8$ 作为输入，使用反函数，$g(8) = \log_{2}(8) = 3$。

所以，$f(3) = 8$，同时 $g(8) = 3$，这证明了这两个函数互为反函数。

相互转换

函数 $2^{\log_2(n)}$ 实际上是一个非常有趣的情况，因为 $\log_2(n)$ 是以 2 为底的对数函数的输出，而 $2^x$ 是以 2 为底的指数函数。

根据指数和对数的性质，$2^{\log_2(n)}$ 实际上等于 $n$。这是因为以 2 为底的对数函数 $\log_2(n)$ 的作用是“消除”指数函数 $2^x$ 的效果，因此结果是 $n$。

所以，$2^{\log_2(n)} = n$。

因数和公因数

在小学数学里，两个正整数相乘，那么这两个数都叫做积的因数，或称为约数。

给定若干个整数，如果有一个(些)数是它们共同的因数，那么这个(些)数就叫做它们的公因数。而全部公因数中最大的那个，称为这些整数的最大公因数。

小学数学定义：假如$a*b=c$（a、b、c都是整数），那么称a和b就是c的因数。

质数和合数

质数是指在大于1的自然数中，除了1和它本身以外不再有其他因数的自然数。

合数指自然数中除了能被1和本身整除外，还能被其他数（0除外）整除的数。

在哈希表中如果能够保证 key 是随机均匀分布的，那么选择质数或者合数作为模数都可以，它们都能输出均匀分布的哈希值。而当 key 的分布存在某种周期性时，对合数取模更容易出现聚集现象。

总而言之，我们通常选取质数作为模数，并且这个质数最好足够大，以尽可能消除周期性模式，提升哈希算法的稳健性。
